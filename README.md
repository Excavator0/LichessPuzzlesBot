# LichessPuzzlesBot
A Telegram bot that sends a random Lichess puzzle.
https://t.me/LichessPuzzlesBot

Type /new_puzzle command to show a random Lichess puzzle from local database.

# Technologies used
- MySQL 8.0
- Hibernate
- TelegramBots library
